## 1. 📝 概述

- 介绍自定义组件 `EnWordList.vue` 的一些基本功能。
- 笔记中的很多展示单词列表的部分，使用的都是这个自定义组件来实现的。

## 2. 📒 单词本组件 `EnWordList.vue` 功能介绍

- 右键词汇展示组件 `EnWordList.vue` 更多功能：
  - ![图 1](https://cdn.jsdelivr.net/gh/Tdahuyou/imgs@main/2025-06-07-10-48-40.png)
- 单词卡片：
  - 单词卡片预加载。
  - Pin 单词卡片。
  - 支持配置单词卡片是否自动弹出。
- 朗读：
  - 单个词汇。
  - 整个单词本的所有词汇。
- 发音：
  - 英式发音
  - 美式发音
- 选中：
  - 全选功能。
  - 全部重置功能。
